# homestay-ci3

Framework used : Codeigniter 3.1.11

CSS Template : SB Admin 2

js: jquery, ajax

plugin js :

- SweatAlert2
- Datepicker
- Datatable
- Chosen
- Daterangepicker
- datetimepicker
- webcamjs

features:

- Dashboard
- Profil
- User
- Monitor

---Data Master---
- Data kamar
- Data pelanggan
- Data sales
- Data pilihan sewa
- Data kategori biaya
- Data jenis barang
- Data sumber biaya
- Data vendor

---Transaksi---
- Pendapatan
- Pembiayaan
- Deposit

---Laporan---
- Laporan Pendapatan per sales
- Laporan Pembiayaan
- Laporan Transaksi
